
# World Cup Simulator (C++)

## Description
A simple C++ console simulation of a World Cup tournament.  
Teams compete in knockout rounds until a final winner is decided randomly.

## Features
- Round of 16
- Quarter Final
- Semi Final
- Final
- Random match simulation

## How to Run

### Compile:
```bash
g++ main.cpp -o worldcup
```

### Run:
```bash
./worldcup
```

(Windows)
```bash
worldcup.exe
```

## Requirements
- C++ compiler (g++ or MSVC)

## Notes
- This version is **not AI-based**
- Matches are random (50/50 chance)
- Simple console output only
